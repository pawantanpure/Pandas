# Introduction to Pandas

## 📖 Overview
This repository contains beginner-friendly examples of the **Pandas** library in Python. It covers the most commonly used Pandas functions for loading, exploring, selecting, and manipulating data.

## 🚀 Topics Covered

- Importing Pandas
- Reading Excel files
- Viewing datasets
- Displaying rows using `head()` and `tail()`
- Dataset information using `info()`
- Statistical summary using `describe()`
- Selecting rows and columns
- Checking missing values
- Data slicing
- Creating new columns
- Dropping columns
- Filtering data
- Creating Pandas Series
- Creating Pandas DataFrames
- Working with dictionaries and lists

## 🛠️ Technologies Used

- Python 3.x
- Pandas
- Jupyter Notebook
